# Chemistry-Infused Bandit Framework for Electrocatalysis

[![Zenodo](https://img.shields.io/badge/Zenodo-10.5281/zenodo.20043537-blue)](https://zenodo.org/)

This repository contains code, data, and plotting assets for the work "Chemistry-infused bandit framework for interpretable autonomous campaign of electrocatalysis" (DOI pending).

The `HER-Optimizer/` folder contains the closed-loop optimization and laboratory hardware-control code used to prepare catalyst recipes, run electrochemical experiments, process HER measurements, and select the next experimental conditions.

The `Data and plot/` folder contains processed campaign data, figure-generation scripts, figure registries, and generated main-text and supporting-information figures for the associated manuscript.
